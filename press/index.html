DuoVox

Download icon

Short Description

Real-time voice translation across 50+ languages. Dual-lane transcription shows both speakers simultaneously.

Long Description

DuoVox translates conversations in real time. It uses dual-lane technology to show both sides of a dialogue simultaneously — each language gets its own panel. With support for 51 languages online and 28 offline, DuoVox works for travellers, professionals, and teams. Features include subtitle export, file translation, a translation glossary, and a zero-retention Private Mode for sensitive conversations.

Key Facts

  • Platforms: Android, Windows
  • Languages: 51 (online), 28 (offline)
  • Pricing: Free, Essential ($35/mo), Premium ($65/mo), Business ($320/mo)
  • Website: duovox.net

QuietTime

Download icon

Short Description

Simple screen time control that respects your family's privacy. No accounts, no cloud, no tracking.

Long Description

QuietTime helps parents manage screen time with scheduled blocking and beautiful animated lock screens. Set separate weekday and weekend schedules, choose from five themes, configure wind-down warnings, and control which apps stay accessible. Everything runs locally on the device — no accounts, no cloud, no tracking, no ads. One-time purchase, yours forever.

Key Facts

  • Platforms: Android, Windows
  • Pricing: £3.49 (Android), £4.49 (Windows) — one-time purchase
  • Privacy: 100% on-device, no data leaves the device
  • Website: davidarthur.app/quiettime
