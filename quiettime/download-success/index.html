Thank You for Your Purchase!

Your download should be available from LemonSqueezy. Below you'll find installation instructions for your platform.

💻 Windows Installation

  1. Download QuietTime_Windows.zip from your purchase confirmation
  2. Extract the zip file to a folder on your computer
  3. Open the extracted folder and run QuietTime.exe
  4. Windows may show a SmartScreen warning — click More info then Run anyway
  5. Create your 6-digit parent PIN when prompted
  6. Set your schedule and you're done!
Tip: Enable "Start with Windows" in Settings so QuietTime runs automatically when the computer boots.

📱 Android Installation

  1. Download QuietTime-release.apk from your purchase confirmation
  2. Open the APK file on your Android device
  3. If prompted, allow installation from unknown sources for your browser or file manager
  4. Tap Install when prompted
  5. Open QuietTime and create your 6-digit parent PIN
  6. Grant the required permissions (Accessibility, Overlay, Device Admin) when prompted
Tip: The permissions are needed to prevent children from closing QuietTime. The app works entirely offline — no data is ever sent anywhere.
